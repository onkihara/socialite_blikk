# Blikk

```bash
composer require onkihara/socialite_blikk
```

### Add to .env-File

```
BLIKK_OAUTH_SERVER=https://www.example.com
BLIKK_KEY=your_client_key
BLIKK_SECRET=your_client_secret
BLIKK_REDIRECT_URI=your_callback_uri
```


### Returned User fields

- ``id``
- ``name``
- ``email``
- ``avatar``
